## Extraction Rules

1. **Do not hallucinate.** Only extract information that is explicitly stated in the transcript.

2. **Post-visit medication list (`ExtractionResult.medications`):**
   - This field must contain ALL medications the patient should be taking AFTER this visit.
   - Start with every medication listed in `patient_info.current_medications` (the intake report).
   - Remove any that the doctor explicitly discontinued.
   - Add any newly prescribed medications.
   - If the doctor prescribes the same drug the patient was already on (possibly at a different dose), include it once with the updated parameters.

3. **Discontinued medications (`ExtractionResult.discontinued_medications`):**
   - Only add a medication here if the doctor EXPLICITLY stops it (e.g. "stop taking X", "discontinue X", "no longer take X").
   - Do NOT add medications that were simply not mentioned — omission does not mean discontinuation.

4. **Current medications at intake (`PatientInfo.current_medications`):**
   - Record only what the patient reports they were already taking before this visit.
   - Do NOT include newly prescribed medications here.

5. **Duration semantics:**
   - If a medication has a specific duration stated (e.g. "for 7 days"), record it.
   - If NO duration is mentioned, leave `duration` as null. Null means the medication is ongoing/indefinite — do NOT invent a default duration.

6. **Conditions vs. symptoms:**
   - `PatientInfo.conditions` = pre-existing diagnosed conditions (e.g. "hypertension", "diabetes").
   - `PatientInfo.symptoms` = what the patient reports during this visit (e.g. "headache", "fever").

7. **Vital signs:**
   - Format as readable strings, e.g. "Blood Pressure: 130/80", "Heart Rate: 72 bpm", "Temperature: 98.6°F".
